# 笔核 APP 技术设计文档

## 1. 项目概述

笔核是一款专为创作者设计的Android原生应用，支持AI辅助续写、漫剧制作、推文视频生成等功能。所有数据本地加密存储，确保用户隐私安全。

## 2. 技术架构

### 2.1 技术栈
- **语言**: Kotlin
- **UI框架**: Jetpack Compose + Material Design 3
- **数据库**: Room (SQLite)
- **网络请求**: Retrofit + OkHttp
- **异步处理**: Coroutines + Flow
- **后台任务**: WorkManager
- **数据存储**: DataStore (SharedPreferences替代)

### 2.2 架构模式
采用MVVM + Clean Architecture架构：
- **UI层**: Compose UI + ViewModel
- **领域层**: UseCase
- **数据层**: Repository + Room Database + Retrofit API

## 3. 模块设计

### 3.1 创作中枢模块
- 项目管理（CRUD）
- 章节管理
- 大纲系统（四级）
- 人物设定管理
- 世界观设定管理

### 3.2 AI续写模块
- DeepSeek API集成
- Prompt模板管理
- 后台续写服务
- 断点续传

### 3.3 漫剧制作模块
- 剧本生成
- 分镜生成
- 配音配乐
- 视频合成

### 3.4 推文视频模块
- 内容导入
- 爽点提取
- 文案生成
- 视频导出

## 4. 数据模型

### 4.1 核心实体
- Project: 项目
- Chapter: 章节
- Volume: 分卷
- Character: 人物
- WorldSetting: 世界观设定
- PlotNode: 剧情节点
- DramaEpisode: 漫剧集数
- Storyboard: 分镜
- PromptTemplate: Prompt模板

## 5. API设计

### 5.1 DeepSeek API
- Base URL: https://api.deepseek.com
- 默认模型: deepseek-chat
- 支持自定义API Key和Base URL

### 5.2 请求格式
```json
{
  "model": "deepseek-chat",
  "messages": [
    {"role": "system", "content": "..."},
    {"role": "user", "content": "..."}
  ],
  "temperature": 0.7,
  "max_tokens": 4096
}
```

## 6. 安全设计

- 所有用户数据使用AES加密存储
- API Key使用Android Keystore保护
- 支持应用密码锁
- 本地备份加密

## 7. 性能优化

- 针对红米12T优化内存使用
- 后台进程保活
- 异步任务调度
- 列表懒加载

## 8. 兼容性

- 最低SDK: Android 7.0 (API 24)
- 目标SDK: Android 14 (API 34)
- 推荐设备: 红米12T (16G+1T)
