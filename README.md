# 笔核 - 个人专属离线创作工具

一款专为创作者设计的Android应用，支持AI辅助续写、漫剧制作、推文视频生成等功能。

## 功能特性

### 📝 创作中枢
- 支持小说、短剧剧本、漫剧剧本、推文文案四类项目
- 四级大纲系统：总纲→分卷大纲→分章章纲→小节节点
- 全格式编辑器，支持多格式导出（TXT/Word/PDF/MD）
- 每5分钟自动本地备份

### 🤖 AI续写
- 集成 DeepSeek API，支持智能续写
- 三种续写模式：章纲定向续写、自由续写、批量多章续写
- 支持锁屏后台运行、断点续传
- 预设5套专用prompt模板

### 🎬 漫剧制作
- 60集漫剧全流程制作
- 自动生成剧本、分镜
- 多角色配音、字幕对齐
- 批量合成9:16竖屏视频

### 🎥 推文视频
- 从小说导入内容
- AI提取爽点生成文案
- 一键合成短视频
- 支持抖音/快手/小红书/视频号格式

### 🔒 安全特性
- 所有数据本地SQLite加密存储
- 应用密码锁
- 一键备份/恢复

## 技术栈

- Kotlin + Jetpack Compose
- Room Database
- Retrofit + OkHttp
- Coroutines + Flow
- WorkManager
- Material Design 3

## 构建

项目使用 Gradle 构建，需要 JDK 17+。

```bash
./gradlew assembleDebug
```
